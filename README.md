# PersonaSEL: AI-Powered Counseling Assistant

**PersonaSEL** adalah chatbot cerdas yang dirancang untuk mendukung layanan Bimbingan dan Konseling (BK) di tingkat SMA. Aplikasi ini mengintegrasikan model **Gemini 1.5 Flash** untuk memberikan respons yang empati, cepat, dan berbasis pada kerangka kerja *Social-Emotional Learning* (SEL).

## 🚀 Fitur Utama
- **Emotional Support:** Membantu siswa mengenali dan meregulasi emosi secara real-time.
- **SEL Integration:** Berpijak pada kerangka kerja CASEL untuk pengembangan diri siswa.
- **Fast Response:** Menggunakan model Gemini 1.5 Flash untuk latensi rendah.
- **Secure Configuration:** Pengelolaan API Key yang aman menggunakan variabel lingkungan.

## 🛠️ Teknologi yang Digunakan
- **Backend:** Node.js, Express.js
- **AI Model:** Google Gemini 1.5 Flash API
- **Frontend:** HTML5, Tailwind CSS (via V0.dev/Shadcn)
- **Environment Management:** Dotenv

## 📋 Prasyarat
Sebelum menjalankan proyek ini, pastikan Anda memiliki:
- Node.js versi 18 atau terbaru.
- API Key dari [Google AI Studio](https://aistudio.google.com/).
